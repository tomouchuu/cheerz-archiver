# cheerz-archiver

## Puppeter Script to Download from Cheerz

(Tampermonkey version available at https://github.com/tomouchuu/cheerz-archiver/tree/main/tampermonkey)

### Install / Run

- Clone the repo
- Within the repo `npm i`
- Run with `node app.js`
